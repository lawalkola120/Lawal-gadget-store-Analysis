# Lawal-gadget-store-Analysis

# Tools used
- power Bi - Data vitualization and Dashboard creation
- DAX - Calcuted measures and KPIs


## Project Overview
This project analyzes sales data from a gadget store to uncover insights about revenue performance, customer distribution, product demand, and weekly sales trends. The dashboard was built to support data-driven decision making by identifying key business metrics and patterns.

## Objectives
	-	Analyze overall sales performance and revenue
	-	Identify best-selling products
	-	Understand customer distribution by region
	-	Track daily sales trends during the week
	-	Explore customer purchasing behavior

 ## Key Metrics
	-	Total Sales Transactions: 200
	-	Total Revenue: 102,501
	-	Total Customers: 20

  ## Dashboard Features
	-	KPI cards showing total sales, revenue, and customers
	-	Product performance analysis highlighting best-selling products
	-	Regional customer distribution visualization
	-	Weekly sales trend analysis
	-	Customer revenue breakdown table
	-	Interactive filters for product category and customer segment
